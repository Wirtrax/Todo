# ✅ Todo App

## 📌 About
Todo App is a fully-featured task management application built with React.  
The project focuses on clean architecture, accessibility (a11y), and performance optimization, following best practices for modern frontend development.

## ✨ Features
- 📝 Create, edit, and delete tasks
- ♿ Full accessibility support (a11y best practices)
- ⚡ Optimized rendering and performance
- 🧩 Well-structured and scalable architecture
- 🎯 Clean and intuitive user interface
- 📱 Responsive design
